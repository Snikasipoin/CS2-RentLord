# CS2 Account Rent Bot

Telegram-бот для управления арендой Steam-аккаунтов с SQLite, aiogram 3 и шифрованием чувствительных данных через Fernet.

## Что умеет сейчас

- Добавлять новые аккаунты с данными Steam, email и Faceit.
- Сохранять ссылку на Faceit-профиль и показывать текущий Elo/уровень в карточке аккаунта.
- Редактировать данные аккаунта прямо из карточки.
- Удалять аккаунт с подтверждением.
- Показывать список аккаунтов и их полный набор данных.
- Сдавать аккаунт в аренду.
- Продлевать аренду, включая минимальный шаг `30 минут`.
- Освобождать аккаунт вручную.
- Показать статус базы:
  - количество свободных аккаунтов;
  - количество занятых аккаунтов;
  - количество свободных аккаунтов с Faceit.
- При сдаче аккаунта отправлять данные покупателю:
  - Steam login/password;
  - Faceit login/password, если они заполнены.
- Создавать резервную копию базы в файл и отправлять её в чат.
- Загружать резервную копию `.db` обратно в бот.
- Хранить `FACEIT API` ключ прямо внутри бота через раздел `💾 Данные`, без необходимости в панели хостинга.

## Меню бота

- `📊 Статус`
- `📦 Аккаунты`
- `➕ Добавить`
- `🎮 Сдать`
- `⏱ Продлить`
- `✅ Освободить`
- `💾 Данные`

## Как работают данные

- Основная база хранится в SQLite-файле `accounts.db`.
- Пароли шифруются через Fernet.
- Faceit API ключ хранится в таблице `settings` и попадает в резервные копии.
- Для хостинга рекомендуется использовать постоянную папку данных:
  - `DATA_DIR=/app/data`

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте `.env` рядом с `bot.py`:

```env
BOT_TOKEN=ваш_токен
ADMIN_ID=ваш_telegram_id
ENCRYPTION_KEY=ваш_fernet_key
```

3. Запустите бота:

```bash
python bot.py
```

## Важные замечания

- `FACEIT_API_KEY` теперь можно вводить прямо через бота в разделе `💾 Данные`.
- Если ключ уже сохранён в боте, он будет использован без необходимости настраивать переменную окружения хостинга.
- Для резервных копий лучше использовать файл, скачанный из раздела `💾 Данные`, а не копировать базу вручную.

## Roadmap

Ближайшие улучшения, которые можно добавить дальше:

- Поиск аккаунтов по логину, email и Faceit-ссылке.
- Фильтры по списку аккаунтов:
  - свободные;
  - занятые;
  - с Faceit;
  - без Faceit.
- История изменений по каждому аккаунту.
- Автоматические бэкапы по расписанию.
- Проверка валидности Faceit API ключа через отдельную кнопку.
- Авто-обновление Elo по расписанию с уведомлением об изменениях.
- Экспорт базы в CSV/Excel.
- Защита от удаления занятых аккаунтов с отдельным принудительным режимом.
- Поддержка нескольких уровней доступа для администраторов.
- Перенос SQLite на более устойчивую БД при росте объёма данных.
